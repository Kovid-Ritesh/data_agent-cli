# DataAgent CLI 📊🤖

DataAgent is a powerful, autonomous AI data analyst that runs directly in your terminal. Powered by Google Gemini, it scans your workspace, auto-loads datasets, and executes sandboxed Python code locally to analyze data, plot charts, and execute system commands safely.

It includes both a fast **Command Line Interface (CLI)** and a sleek, responsive **Web Dashboard** served locally via FastAPI.

---

## Key Features 🚀

- 🔒 **100% Local Sandboxing**: Code generated by the AI is executed within a secure, isolated local Python sandbox. Your private datasets never leave your machine.
- 📂 **Zero-Config Auto-Discovery**: Launch `dagent` in any folder, and it automatically scans, loads, and extracts schemas for up to 10 CSV, Excel, TSV, Parquet, JSON, or SQLite files.
- 🔄 **Self-Correcting ReAct Loop**: If the generated Python script fails or requests a missing module, the agent captures the traceback, automatically run packages install, and resolves the issue on the fly.
- 💻 **Hybrid CLI & Web UI**:
  - Run the agent globally inside your terminal using `dagent`.
  - Or, launch the web dashboard server to upload files and chat interactively.
- 🎨 **Rich Terminal Rendering**: Beautiful markdown formatting, color-coded thinking phases, code execution boxes, and tabulate-rendered dataframes.

## Getting Started & Installation 📦

### Option 1: Standalone Binary (Recommended — No Python Required)

#### Windows (PowerShell)
To install the standalone `dagent.exe` binary and register it in your User PATH automatically, run:
```powershell
powershell -c "irm https://kovid-ritesh.github.io/data_agent-cli/install.ps1 | iex"
```
*Alternatively, download and double-click the [dagent-installer.bat](https://kovid-ritesh.github.io/data_agent-cli/dagent-installer.bat) script.*

#### macOS / Linux
To download the executable and register the path, run:
```bash
curl -fsSL https://kovid-ritesh.github.io/data_agent-cli/install.sh | sh
```

---

### Option 2: Python Package (Pip)

Ensure you have Python 3.9+ installed, then run:
```bash
pip install git+https://github.com/Kovid-Ritesh/data_agent-cli.git
```

## Configuration: Gemini API Key 🔑

DataAgent runs under your own Google AI Studio Gemini API key. This guarantees privacy and zero server costs.

1. Get a free API key from the **[Google AI Studio](https://aistudio.google.com/)** dashboard.
2. Add the API key to your environment variables:

**Windows (PowerShell)**:
```powershell
$env:GEMINI_API_KEY="your_api_key_here"
```

**macOS / Linux**:
```bash
export GEMINI_API_KEY="your_api_key_here"
```

**Using `.env` file**:
Alternatively, create a file named `.env` in the directory you run the command from, or in the package directory:
```env
GEMINI_API_KEY=your_api_key_here
```

## Usage Guide 📖

Once installed and configured, open your terminal in any directory containing data files (e.g., `sales.csv`).

### 1. Interactive Mode
Launch the interactive REPL shell:
```bash
dagent
```
Ask questions directly inside the prompt:
```text
[DataAgent] > Show a summary of revenue by region and plot a bar chart.
```

### 2. Inline / Single-Shot Mode
Execute queries directly as command arguments and exit:
```bash
dagent "Summarize sales.csv and write a report of top reps"
```

### 3. Special Shell Commands
Inside the interactive prompt, you can use the following commands:
- `files` : Lists all currently loaded dataframes, their row/column dimensions, and sizes.
- `load <path>` : Loads a new dataset dynamically (supports drag-and-dropping files directly).
- `unload <name>` : Unloads a dataset from the agent's memory.
- `clear` : Clears conversation memory history.
- `cls` : Clears the terminal screen.
- `exit` or `quit` : Exits the agent shell.

## Web Dashboard Server 🌐

DataAgent also includes a FastAPI server that hosts a gorgeous web application interface.

To start the server:
```bash
python server.py
```

- **Landing Page**: Visit `http://localhost:8000/` to read documentation, download installers, and explore setup instructions.
- **Interactive Web App**: Visit `http://localhost:8000/app` to upload data files, chat with the analyst, view code trace logs, and display generated charts in real-time.

## Manual Execution & Standalone Compilation 🛠️

If you are developing locally or want to compile the standalone binary yourself:

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Run from Source
```bash
python main.py
```

### 3. Compile the Standalone Binary
To compile `dagent.exe` or `dagent` using PyInstaller:
```bash
pyinstaller dagent.spec
```
*Note: The `dagent.spec` config is pre-optimized to exclude heavy scientific modules (like Torch, SciPy, pytest) to keep the compiled binary compile time and size minimal.*
