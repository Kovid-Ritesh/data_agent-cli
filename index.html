An Autonomous AI Data Analyst
Directly in Your Terminal

DataAgent is a local CLI agent that scans your folders, loads your datasets, and runs sandboxed Python code to analyze data, plot charts, and execute system tasks. Fully private, fully autonomous, and runs in seconds.

Launch Web App Get Standalone Executable
$ pip install git+https://github.com/your-username/dataagent-cli.git

Built for Speed, Privacy, and Power

DataAgent runs entirely inside your terminal session. It does not use external web servers, keeping your datasets secure.

100% Local Sandboxing

All Python code is executed inside a persistent local sandbox on your machine. Your private data is never sent to third-party databases.

Zero Configuration Startup

Launch the tool in any folder. It automatically auto-discovers and loads up to 10 CSV, TSV, Parquet, Excel, or SQLite files in seconds without blocking prompts.

Self-Correcting Code Loop

If Python code throws errors or missing packages are requested, the agent dynamically catches the traceback, runs pip installs, and self-corrects on the fly.

Autonomous CLI Power

The agent can run terminal commands (like listing directories or copying files) with your interactive approval to automate tedious developer workflows.

Watch the ReAct Loop in Action

Get Started in Seconds

Choose the installation method that fits your environment.

1

One-Click Terminal Installation (Recommended)

Open your terminal and run the command to download, install, and add the agent to your system Path automatically:

Windows (PowerShell)
> powershell -c "irm https://raw.githubusercontent.com/your-username/dataagent-cli/main/website/install.ps1 | iex"
Or download double-click installer: dagent-installer.bat
macOS / Linux (Shell)
$ curl -fsSL https://raw.githubusercontent.com/your-username/dataagent-cli/main/website/install.sh | sh
Or download installer script: dagent-installer.sh
2

Alternative: Manual Download

If you prefer a manual setup, download the pre-compiled `dagent.exe` executable and copy it to any directory on your system PATH:

Download dagent.exe
3

Verify Installation

Open a NEW Command Prompt (CMD) or PowerShell window in any directory containing data files and type:

$ dagent
1

Install Package via Git

Ensure you have Python 3.9+ installed, then run the pip command to download and build the script globally:

$ pip install git+https://github.com/your-username/dataagent-cli.git
2

Run in Interactive Mode

Open your command line in any directory containing files and run:

$ dagent
3

Execute Single Queries (Inline Mode)

You can also execute questions directly from terminal parameters and exit:

$ dagent "Read sales.csv and output a summary of region vs revenue"

Configure Your Google Gemini API Key

DataAgent requires your own API key to make reasoning requests to Gemini. This keeps your costs at zero and ensures your key is never shared.

1

Visit Google AI Studio

Go to the official Google AI Studio dashboard and log in with your normal Google account.

2

Create Key

Click the blue **"Get API Key"** button in the top left, then select **"Create API Key in new project"**. Copy the generated key token.

3

Set Environment Variable

Set it globally in your terminal. For example, in PowerShell:

$ $env:GEMINI_API_KEY="your_api_key_here"

Or create a file named `.env` containing `GEMINI_API_KEY=your_key` in the folder you run `dagent` from.
